# 2024 Global Sales Data Analysis

Comprehensive monthly sales dataset for **5 segments**, **5 countries**, multiple products, and managers. Ideal for portfolio projects, dashboards, and learning data analysis.

## Dataset Overview
- **Time Period**: Jan–Dec 2024 (12 files + consolidated Book1.xlsx)
- **Rows**: \~400+ per month
- **Key Columns**: Segment, Country, Product, Discount Band, Manager, Units Sold, Manufacturing Price, Sale Price, Gross Sales, Discounts, Sales, COGS, Profit

## Key Insights
- Total Revenue: $99,370,811
- Top Performing Product/Segment/Country: Paseo,Government,France...
- Profit Margin Trends: Enterprise jas noticeably lower Gross Profit..
